Optimization

移动端性能优化不是一组技巧，而是一条预算链

把 CPU、GPU、内存和 IO 拆成预算表，再做对象池、批处理和异步加载，定位会比“感觉卡”更准确。

中文 UI 显示异常时，先查布局，再查字体

很多所谓“字体缺字”其实是文字容器宽度、换行模式、图标占位和平台回退字体共同作用的结果。

SDK 能出广告不代表配置已经完整，app-ads.txt、应用关联、商店信息和回调监控都要拉通。

Toolkit

实用专题

角色与战斗

把角色控制拆成输入、运动、状态机和伤害反馈四层，调试时才不会把 bug 混在一起。

背包与存档

拖拽交互、双击装备、战局内临时收益与持久化边界，要在设计阶段就分清楚。

移动 UI

同一组文案在不同机型上看起来完全正常，不代表极窄屏、小字号和回退字体也稳定。

商业化配置

广告收入问题很多都不是 SDK 接入错误，而是商店信息、域名验证和政策配置没有闭环。
